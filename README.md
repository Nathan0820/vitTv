# Welcome to VitTv!

A retro-style TV interface built with React and Vite. Watch vintage shows or commercials by changing channels on a vintage TV set.

## Features

- Retro TV design with vintage aesthetics
- Channel switching to browse different YouTube videos
- Power on/off functionality
- Responsive design
